# Claude 명령어(Command) 정리

## 1. 명령어 목록

| 명령어 | 주요 기능 | 사용 예시 | 주의사항 및 팁 |
|--------|----------|-----------|---------------|
| `/init` | 프로젝트 초기화 및 설정 | `/init` → 전체 폴더 구조를 분석하고 CLAUDE.md 자동 생성 | Claude가 파일 관계, 폴더 구조, 주요 코드를 파악해 프로젝트 맥락 설명 파일인 CLAUDE.md 1개 생성 및 관찰 |
| `/clear` | 대화 및 컨텍스트 초기화 | `/clear` | Claude의 기억 대화 히스토리를 모두 제거함. 이전 코드 내용이나 프로젝트 상태를 완전히 잊음 |
| `/model <모델명>` | AI 모델 변경 | `/model opus` 또는 `/model sonnet` | Opus = 고정밀 느림 / Sonnet = 속도 우선. 상황에 맞게 선택 |
| `/compact "<요약문>"` | 대화 내용을 요약하여 토큰 절약 | `/compact "이전 대화 요약문"` | 대화가 길어질 때 실행 지시하게 자주 사용하면 맥락 손실 발생 가능 |
| `/cost` | API 토큰 사용량 확인 | `/cost` | Opus 사용 시 비용이 높음. 경기 세심하게 서는 추가할 확인 필요 |
| `/memory` | CLAUDE.md 파일 열기 및 수정 | `/memory` | Claude가 학습 중인 프로젝트 메모리 직접 열람/편집 가능 |
| `/help` | 전체 명령어 목록 표시 | `/help` | 모든 명령어와 요약된 설명을 한 번에 확인 |
| `/permissions` | 작업 권한 관리 | `/permissions` | Auto-Accept 모드로 함께 사용률 대 유용. 권한 부여 시 주의 필요 |
| `/commands` | 커스텀 명령어 목록 표시 | `/commands` | `.claude/commands/` 폴더 내 사용자 정의 명령어 확인 |

## 2. 커스텀 명령어

### 반복 작업 자동화

`.claude/commands/test.md` 파일 생성 후 내용 작성 시

```markdown
$ARGUMENTS 변수를 사용해 명령어 실행 시 인자 전달 가능
```

예시: 테스트 실행, 빌드 자동화 등

## 3. 주요 명령어 상세 설명

### `/init` - 프로젝트 초기화
- **기능**: Claude가 프로젝트 전체 구조를 분석하여 CLAUDE.md 파일을 자동 생성
- **사용 시기**: 
  - 새 프로젝트 시작 시
  - Claude에게 프로젝트 맥락을 처음 알려줄 때
  - 프로젝트 구조가 크게 변경되었을 때
- **생성 파일**: CLAUDE.md (프로젝트 전체 구조, 주요 파일, 의존성 등 정리)

### `/compact "<요약문>"` - 대화 압축
- **기능**: 긴 대화 내용을 요약하여 토큰 사용량을 줄임
- **사용 시기**:
  - 대화가 너무 길어져서 컨텍스트 제한에 도달할 때
  - 이전 대화 내용은 유지하되 토큰을 절약하고 싶을 때
- **주의사항**:
  - 너무 자주 사용하면 세부 맥락 손실 가능
  - 중요한 코드나 결정사항은 요약문에 명시적으로 포함
- **사용 예시**:
  ```
  /compact "지금까지 로그인 API 구현 완료. JWT 토큰 방식 사용. 다음은 회원가입 기능 구현 예정"
  ```

## 4. 사용 팁

### 효율적인 작업 흐름
1. **프로젝트 시작**: `/init`으로 초기화
2. **작업 진행**: 일반 대화로 개발 진행
3. **대화가 길어질 때**: `/compact`로 요약
4. **새로운 주제 시작**: `/clear`로 리셋 (필요시)
5. **비용 확인**: `/cost`로 토큰 사용량 체크

### 모델 선택 가이드
- **Opus**: 복잡한 아키텍처 설계, 중요한 버그 수정
- **Sonnet**: 일반적인 코드 작성, 빠른 반복 작업

### 권한 관리
- `/permissions`로 Auto-Accept 모드 설정 시 Claude가 자동으로 파일 생성/수정 가능
- 중요한 파일 작업 시에는 수동 승인 권장

---

**참고**: 이 명령어들은 Claude Code 또는 Claude.ai의 특정 환경에서 사용 가능합니다.
